# IServ MacOS App 

---

A native macOS application designed to make accessing your IServ portal faster, easier, and more convenient. No more cluttered browser tabs or progressive-web-app-installations – keep your tasks and emails right in your Dock.

###  Features

* **Quick Access:** Starts as a standalone app, eliminating browser clutter.
* **Native Integration:** Leverages the macOS system for a smooth experience.
* **Lightweight:** Consumes fewer resources than a browser with dozens of open tabs.

###  Installation

To install the app, follow these simple steps:

1. Go to the [Releases](https://github.com/FarinAlten/IServ-MacOS-APP/releases) page.
2. Download the latest `.dmg` file.
3. Open the `.dmg` file by double-clicking it.
4. Drag the **IServ App** into your **Applications** folder.
5. Launch the app via Launchpad or the Applications folder.

> [!IMPORTANT]
> **Note for the first run:** Since the app is not (yet) signed by Apple, you might need to start it for the first time by  getting into the scurity section in settings and and allow the app to run on your mac!

###  Usage

After launching the app, you will be prompted to enter your IServ instance (e.g., `your-school.com`). Simply log in with your usual credentials. The app remembers your session so you don't have to log in every time.
